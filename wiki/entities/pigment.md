# Pigment

Enterprise financial planning platform. Implemented a production Supervisor Agent multi-agent architecture with domain expert agents. Published the "Building a Hivemind" article on multi-agent system design.

## Implementation

Supervisor formation:
- Supervisor Agent — coordinates operations and user communication
- Analyst Agent, Planner Agent, Modeler Agent — domain experts
- Reporter Agent — generates and publishes charts/data

## Customer Results

- Supercell: P&L updates from 8 days → 4 minutes
- Carta: 80% time reduction on data aggregation
- Fivetran: 12 hours/month saved on executive reporting
- Evenflo: 6 days faster scenario creation
