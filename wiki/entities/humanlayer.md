# HumanLayer

Company building Human-in-the-Loop tooling for AI agents. Founded by [[Dex Horthy]]. Also the organization behind the [[12-Factor Agents]] open-source framework and the kubechain project (distributed agents in Kubernetes).

## Products / Projects

- [[12-Factor Agents]] — open-source engineering principles for production LLM software
- HumanLayer SDK — tooling for integrating human approval gates into agent workflows
- kubechain — framework for running distributed agents in Kubernetes
