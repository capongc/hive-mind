# Claude Code

Anthropic's agentic coding tool — a CLI and IDE integration that operates as a Developer Agent in an [[Agent Swarm]]. Used by developer agents for implementation within VS Code and other environments.

## Role in the AI-Native SDLC

In the [[AI-Native SDLC]], Claude Code is the implementation engine used by Developer Agents. It handles the "bounce" in the [[V-Bounce Model]] — generating code near-instantly from design specs provided by the [[Engineering Lead Role]].

## Industry Signal

[[Ramp]] stated: *"If you're not using Claude Code, no matter what your role is, you're probably underperforming."* — positioning Claude Code proficiency as a baseline performance expectation.
