# LangGraph

A graph-based multi-agent orchestration framework from LangChain. Models agent workflows as directed graphs — nodes are agents or tools, edges are transitions — enabling complex, stateful multi-agent pipelines.

## Use in AI-Native Teams

LangGraph is an alternative to [[CrewAI]] for structuring an [[Agent Swarm]]. Its graph model is particularly well-suited for workflows with conditional branching, cycles, or complex state management across agents.
