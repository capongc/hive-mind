# CrewAI

A Python-based multi-agent orchestration framework for structuring virtual AI teams. Agents are organized into "crews" with defined roles, goals, and tools — enabling role-based delegation and collaboration across the [[Agent Swarm]].

## Use in AI-Native Teams

CrewAI is one of the primary frameworks for composing an [[AI-Native SDLC]] virtual team. A typical crew might include:
- [[Engineering Lead Role]]
- Developer Agents
- Test Engineer Agent
